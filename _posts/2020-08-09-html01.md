## 1. HTML 태그
```html
<meta http-equiv='refresh' content='0;url=이동할주소'>
```
0초 후 url에 있는 주소로 즉시 이동한다. <br>
#### 예시
```html
<meta http-equiv="refresh" content="0;URL='http://example.com/'">
```
```html
<meta http-equiv='refresh' content='0;url=/wiki'>
```
<br>

## 1. 자바스크립트
```html
<script>
location.href="이동할주소";
</script>
```
